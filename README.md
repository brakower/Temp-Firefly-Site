# Firefly Propane | Coming Soon

Placeholder site for the Firefly Propane rebrand. Includes a coming soon message, appointment phone number, services list, locations, and supporting details. Mock data is included and should be replaced with verified information.

## Requirements

- Node.js 20.19+ (or 22.12+)

## Commands

- Install dependencies: `npm install`
- Run the dev server: `npm run dev`
- Build for production: `npm run build`

## Update Content

Edit the content in `src/App.jsx` and the styles in `src/App.css` and `src/index.css`.
